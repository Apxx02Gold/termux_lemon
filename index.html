Instalar L3MON en Termux

Una solución rápida para instalar lemon

A continuación les mostrare los pasos a seguir para instalar L3MOM correctamente 🐙

REQUISITOS :

  • apkmod ( Instalar desde aquí> )

  • Paquete Lemon Deb (Obtener en el Telegram oficial)

Pasos de instalación :

  1. Descarga el paquete lemon deb del telegram oficial

  2. Luego instale apkmod

  3. Instale la versión anterior de nodejs y npm 
    apt install nodejs && npm install -g npm@6.14.10
  4. Ahora instale el paquete deb 
    dpkg -i lemon_2.1-2_all.deb

CONSEJOS :

  • Para iniciar el servidor

      lemon

  • Si inició el servidor por primera vez, presione CTRL+C y edite el archivo maindb.json

      vi $PREFIX/share/lemon/server/maindb.json

    Establezca la contraseña en Md5 Ejemplo:098f6bcd4621d373cade4e832627b4f6 peguelo en el campo de password 👇🏽

  • Ahora guarde e inicie el servidor usando el comando lemon en el navegador escriba 127.0.0.1:22533 desde el navegador escriba usuario y contraseña establecida
